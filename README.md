# kotivas.github.io

Personal portfolio site built with [Astro](https://astro.build), featuring a WebGL background powered by [ogl](https://github.com/oframe/ogl)

## Stack

- Astro
- ogl (WebGL2)
- vite-plugin-glsl

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is generated in `dist/`.
